# Pokemon Coding Task

The app consist of:
* Navigation
* A view showing a list of pokemons
* A detailed view showing information about the chosen pokemon from the list

## Features

App features:
* `MVVM architecture`
* `SwiftUI & Combine`
* `Dark Mode Supported`

## Requirements

- iOS 12.0+
- Xcode 14.x
